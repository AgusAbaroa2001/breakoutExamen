# breakoutExamen
JUEGO BREAKOUT
Realizado por Agustin Abaroa y Alan Agundez

---------------------------------------------------------
CONTROLES:
Izquierda: Flecha Izquierda     Derecha: Flecha Derecha
Lanzar Bola: Barra Espaciadora
---------------------------------------------------------
-------------------------------------------------------------------------------------------
INSTRUCCIONES:
1.- El juego iniciara directamente pero al bola se mantendra estatica hasta que la lances
2. Rompe todas las barras para pasar de nivel
3. Para obtener puntos debes romper una barra (+1 punto por cada barra)
4. Contaras con 3 vidas, cada vez que la bola caiga mas alla del suelo perderas una vida
y si pierdes las 3 vidas el juego se reiniciara
5. Cuando rompas todas las barras de un nivel pasaras la siguiente
-------------------------------------------------------------------------------------------
